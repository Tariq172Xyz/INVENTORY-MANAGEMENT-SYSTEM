# Inventory and Sales Management System

## Description
This is a simple Inventory and Sales Management System built in Java. It helps businesses manage products, track sales, and keep stock updated automatically.

## Features
- Add, update, delete, and view products.
- Record sales and calculate total sales.
- Track stock levels and low-stock items.
- Generate sales and inventory reports.

## Technologies Used
- Java
- MySQL
- JDBC

## Setup
1. Install Java and MySQL.
2. Import the provided database script.
3. Update database connection in DBConnection.java.
4. Run the application.
